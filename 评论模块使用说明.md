# 客户评论滚动横幅模块使用说明

## 📁 文件说明

1. **alibaba-reviews-code.html** ⭐ **（重点文件）**
   - 阿里国际站专用代码
   - 可直接复制粘贴使用
   - 包含自动滚动和悬停放大效果

2. **customer-reviews.html**
   - 完整的HTML页面
   - 可在浏览器中直接打开预览
   - 用于本地测试效果

## 🎯 核心功能

✅ **上下两行滚动**
   - 第一行：从左到右自动滚动
   - 第二行：从右到左自动滚动

✅ **鼠标交互效果**
   - 鼠标悬停：滚动暂停
   - 同时放大 1.15 倍
   - 增强阴影和边框效果

✅ **无缝循环滚动**
   - 评论卡片循环播放
   - 永不停止

✅ **商务大气蓝主题**
   - 与公司简介板块颜色一致
   - 渐变蓝色标题背景

## 🚀 快速使用步骤

### 步骤 1: 复制代码
打开 `alibaba-reviews-code.html` 文件，复制**全部代码**（包括 `<style>` 标签）

### 步骤 2: 粘贴到阿里国际站
1. 登录阿里国际站后台
2. 进入产品详情编辑页面
3. 找到"源代码"或"HTML编辑"按钮
4. 点击切换到源代码模式
5. 粘贴全部代码
6. 点击保存

### 步骤 3: 预览效果
- 点击预览按钮查看滚动效果
- 将鼠标移到评论卡片上测试悬停效果
- 确认无误后发布

## ✏️ 内容修改指南

### 1. 修改标题

找到标题部分的代码：
```html
<h1>Customer Reviews</h1>
<p>Part of positive reviews</p>
```

可以修改为：
```html
<h1>客户评价</h1>
<p>真实好评展示</p>
```

### 2. 修改评论内容

每个评论卡片的结构如下：

```html
<div class="review-card-scrolling">
    <div class="review-header-scrolling">
        <div class="reviewer-avatar-scrolling">M</div>  <!-- 用户头像字母 -->
        <div class="reviewer-info-scrolling">
            <div class="reviewer-name-scrolling">M***t</div>  <!-- 用户名 -->
            <div class="reviewer-country-scrolling">🇪🇬 Egypt</div>  <!-- 国家 -->
            <div class="verified-badge-scrolling">✓ Verified purchase</div>
        </div>
        <div class="review-date-scrolling">Jun 27, 2025</div>  <!-- 日期 -->
    </div>
    <div class="star-rating-scrolling">★★★★★</div>  <!-- 星级 -->
    <div class="review-content-scrolling">
        评论内容文字...  <!-- 评论内容 -->
    </div>
    <div class="product-info-scrolling">
        <div class="product-image-scrolling"></div>
        <div class="product-details-scrolling">
            <div class="product-name-scrolling">产品名称...</div>  <!-- 产品名 -->
            <div class="product-price-scrolling">$375-385</div>  <!-- 价格 -->
        </div>
    </div>
</div>
```

直接修改对应的文字内容即可。

### 3. 添加更多评论

1. 复制整个评论卡片代码块（从 `<div class="review-card-scrolling">` 到 `</div>`）
2. 粘贴到对应行的滚动轨道内
3. 修改内容
4. **重要**：为了实现无缝滚动，需要在末尾再复制一份所有评论

### 4. 删除评论

直接删除对应的评论卡片代码块，注意保持成对出现（因为无缝滚动需要复制一份）

### 5. 修改头像颜色

## 🎨 样式修改指南

### 1. 修改滚动速度

找到 CSS 中的动画时长：

```css
/* 第一行 - 从左到右 */
.scroll-row-reviews.left-to-right .scroll-track-reviews {
    animation-duration: 60s;  /* 60秒完成一次滚动，数值越小越快 */
}

/* 第二行 - 从右到左 */
.scroll-row-reviews.right-to-left .scroll-track-reviews {
    animation-duration: 60s;  /* 可以设置不同的速度 */
}
```

推荐速度：
- 慢速：80s - 100s
- 中速：60s（默认）
- 快速：30s - 40s

### 2. 修改放大倍数

找到悬停效果的代码：

```css
.review-card-scrolling:hover {
    transform: scale(1.15);  /* 1.15 = 放大 15%，可以改为 1.2、1.3 等 */
}
```

推荐倍数：
- 轻微放大：1.1
- 适中放大：1.15（默认）
- 明显放大：1.2 - 1.3

### 3. 修改卡片宽度

```css
.review-card-scrolling {
    min-width: 450px;  /* 修改这个数值改变卡片宽度 */
}
```

### 4. 修改主题色

将所有 `#2874c9`（蓝色）替换为其他颜色：

```css
/* 例如改为绿色主题 */
color: #27ae60;  /* 绿色 */
border-color: #27ae60;

/* 标题背景渐变也要相应修改 */
background: linear-gradient(135deg, #1e8449, #27ae60, #186a3b);
```

### 5. 调整卡片间距

```css
.review-card-scrolling {
    margin: 0 15px;  /* 左右间距，数值越大间距越大 */
}
```

### 6. 修改行间距

```css
.scroll-row-reviews {
    margin-bottom: 30px;  /* 两行之间的间距 */
}
```

## 💡 高级定制

### 1. 添加产品图片

将产品图片背景色改为实际图片：

```html
<!-- 之前 -->
<div class="product-image-scrolling"></div>

<!-- 之后 -->
<div class="product-image-scrolling" style="background-image: url('您的图片链接'); background-size: cover; background-position: center;"></div>
```

### 2. 调整星级评分

```html
<!-- 5星 -->
<div class="star-rating-scrolling">★★★★★</div>

<!-- 4星 -->
<div class="star-rating-scrolling">★★★★☆</div>

<!-- 3星 -->
<div class="star-rating-scrolling">★★★☆☆</div>
```

### 3. 暂停和恢复滚动

如果想要点击暂停，可以添加 JavaScript：

```javascript
<script>
document.querySelectorAll('.review-card-scrolling').forEach(card => {
    card.addEventListener('click', function() {
        const track = this.closest('.scroll-track-reviews');
        if (track.style.animationPlayState === 'paused') {
            track.style.animationPlayState = 'running';
        } else {
            track.style.animationPlayState = 'paused';
        }
    });
});
</script>
```

### 4. 修改渐变遮罩宽度

边缘渐变遮罩效果：

```css
.scroll-row-reviews::before,
.scroll-row-reviews::after {
    width: 100px;  /* 修改遮罩宽度，0 则不显示遮罩 */
}
```

## 🔧 常见问题

### Q1: 滚动不流畅怎么办？
**A:**
- 减少评论卡片数量
- 增加动画时长（如 60s 改为 80s）
- 确保浏览器支持 CSS3 动画

### Q2: 悬停后不放大？
**A:**
- 检查是否在源代码模式粘贴
- 确认 `<style>` 标签是否完整
- 清除浏览器缓存后重试

### Q3: 手机端显示异常？
**A:**
- 卡片宽度可能过大，建议使用媒体查询：
```css
@media (max-width: 768px) {
    .review-card-scrolling {
        min-width: 300px;
    }
}
```

### Q4: 滚动中间有间断？
**A:**
- 确保评论卡片被复制了一份
- 检查两份内容是否完全一致

### Q5: 想要三行滚动怎么办？
**A:**
复制整个 `<div class="scroll-row-reviews...">...</div>` 代码块，粘贴到第二行下方，并修改为：
```html
<div class="scroll-row-reviews left-to-right">
    <!-- 第三行，从左到右 -->
</div>
```

### Q6: 如何添加真实客户截图？
**A:**
可以将整个评论卡片改为图片：
```html
<div class="review-card-scrolling" style="padding: 0; overflow: hidden;">
    <img src="客户评论截图链接" alt="Customer Review" style="width: 100%; height: auto; display: block;">
</div>
```
