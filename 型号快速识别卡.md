## 一眼识别表

| 看到这些前缀 | 就是这类产品 | 举例 |
|-------------|-------------|------|
| **M393, M386, HMA, HMAA, MTA, KSM, KTH** | 🔵 服务器内存 | M393A4K40CB2-CTD |
| **ST, WD, HU, AL, MG** | 🟠 机械硬盘 HDD | ST1200MM0009 |
| **MZ, PM, SSD, MTF, WDBY** | 🟢 固态硬盘 SSD | MZ7LH960HAJR |
| **Xeon, EPYC, E5-, E7-** | 🔴 CPU处理器 | Xeon Gold 6248R |
| **9361, 9460, PERC, P408, P816** | 🟣 RAID卡 | PERC H740P |
| **QLE, LPe, 9400-8i, 9500** | ⚪ HBA卡 | QLE2692 |
| **X710, E810, MCX, ConnectX** | 🟤 网卡 | X710-DA2 |
| **PWS, xxxW, PSU** | ⚫ 电源 | PWS-1K28P-SQ |

## 内存品牌速查

| 前缀 | 品牌 |
|------|------|
| M393, M386 | Samsung 三星 |
| HMA, HMAA, HMT | SK Hynix 海力士 |
| MTA | Micron 美光 |
| KSM, KTH, KTD | Kingston 金士顿 |

### DDR代数
- **PC3-xxxxx** = DDR3
- **PC4-xxxxx** = DDR4
- **PC5-xxxxx** = DDR5

| HU | HGST |
| AL, MG | Toshiba 东芝 |

### SSD固态盘
| 前缀 | 品牌 |
|------|------|
| MZ, PM | Samsung 三星 |
| SSD, SSDPE | Intel 英特尔 |
| MTF | Micron 美光 |

### 尺寸判断
- **2.5" SFF**: 型号含 MM, MP, SFF
- **3.5" LFF**: 型号含 NM, LFF 或无特殊标记

## 贸易条款一句话

| 条款 | 记忆要点 |
|------|---------|
| **EXW** | 工厂交货，运费客户出 |
| **FOB** | 含出口清关，运费客户出 |
| **CIF** | 含运费保险，关税客户出 |
| **DDP** | 门到门全包，含关税 |
