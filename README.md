# JC 学习推送系统

定时学习推送软件，专为服务器产品外贸业务设计。

## 功能特点

- **定时推送**：可设置多个推送时间点（最多5个），到时间自动推送学习内容
- **灵活设置**：随时更改推送时间和学习类别
- **桌面通知**：支持系统桌面通知和浏览器通知
- **进度追踪**：记录学习进度，统计已完成课程

## 学习内容

### 1. 服务器产品知识
基于您的产品线，包含以下模块：

- **Server Memory (服务器内存)**: DDR3, DDR4, DDR5
- **Server Processor (服务器处理器)**: CPU, Heatsink and Fan
- **Server Hard Drive (服务器硬盘)**:
  - HDD: 2.5in SFF, 3.5in LFF, MSA HDD, 3PAR HDD, EMC HDD, NetApp HDD
  - SSD
  - Caddy/Tray/Casing
- **Server Power Supply (服务器电源)**
- **Server Card (服务器卡)**: RAID Card, HBA Card (Network Card, FC Card)
- **Server Motherboard (服务器主板)**

每个产品课程包含：
- 产品基础知识
- 技术参数详解
- 英文专业术语
- 选型指南

### 2. 外贸练习
围绕服务器产品设计的外贸实战练习：

- **询盘练习**: 询盘分析、回复技巧、专业模板
- **信息回复**: 常见问题应答、议价处理、技术问题解答
- **邮件撰写**: 开发信、报价邮件、跟进邮件
- **订单转化**: 提高转化率技巧、处理客户顾虑、订单确认流程
- **付款催款**: 付款方式介绍、催款邮件技巧
- **售后服务**: 发货通知、质量投诉处理、RMA流程、客户关系维护

### 3. 阿里国际站运营
实用的国际站运营知识与技巧：

- **自然流量提升**: 流量结构分析、曝光提升方法
- **搜索排名优化**: 排名机制、关键词挖掘与布局、CTR提升
- **产品优化**: 信息完整度、主图优化、详情页优化
- **产品等级提升**:
  - 普通品 → 潜力优品
  - 潜力优品 → 优品
  - 优品 → 爆品
- **实战分析模板**: 店铺数据分析、竞品分析、行动计划

## 安装与运行

### 环境要求
- Node.js 16+
- npm 或 yarn

### 安装步骤

```bash
# 1. 安装依赖
npm install

# 2. 启动服务
npm start

# 3. 访问应用
# 打开浏览器访问 http://localhost:3000
```

### 开发模式
```bash
npm run dev  # 支持热重载
```

## 使用说明

### 设置推送时间
1. 点击右上角"设置"按钮
2. 添加或修改推送时间点
3. 选择要推送的学习类别
4. 保存设置

### 学习流程
1. 系统会在设定时间推送学习内容
2. 也可以点击"立即推送"手动获取内容
3. 学习完成后点击"标记完成"记录进度
4. 可以浏览全部课程，自主选择学习

### 实时分析功能
您可以随时发送阿里国际站的实际数据给我分析：
- 数据管家截图
- 产品信息截图
- 询盘截图
- 关键词报告

我会根据您的实际情况提供：
- 现状分析
- 问题诊断
- 优化建议
- 具体行动步骤
