## 文档目录

### 📘 [服务器配件贸易完整指南](./服务器配件贸易完整指南.md)

详细的学习手册，包含：

1. **产品型号识别指南**
   - 服务器内存 (DDR3/DDR4/DDR5, RDIMM/LRDIMM)
   - 服务器硬盘 (HDD/SSD, SATA/SAS/NVMe)
   - 服务器处理器 (Intel Xeon/AMD EPYC)
   - 服务器主板、RAID卡、HBA卡、网卡、电源

2. **报价所需信息清单**
   - 产品基本信息确认
   - 不同产品的额外确认项

3. **国际贸易条款详解 (Incoterms 2020)**
   - EXW/FOB/CIF/DDP等常用条款
   - 选择条款的决策指南

4. **空运快递公司及区别**
   - DHL/FedEx/UPS/TNT对比
   - 运费计算方式
   - 货代专线选择

5. **完整贸易流程**
   - 询价→报价→订单→付款→发货→售后

6. **实用工具与资源**
   - 型号查询网站
   - 邮件沟通模板

## 快速开始

1. 先阅读[完整指南](./服务器配件贸易完整指南.md)了解基础知识
2. 打印[快速识别卡](./型号快速识别卡.md)放在工位
3. 遇到不认识的型号，按前缀规则判断产品类型
4. 报价时使用清单确认所有必要信息
