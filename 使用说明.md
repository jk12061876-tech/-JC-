# 阿里国际站公司详情板块使用说明

## 📁 文件说明

1. **alibaba-code.html** - 阿里国际站专用代码（推荐使用）
   - 可直接复制到阿里国际站编辑器
   - 无需修改，开箱即用

2. **company-profile.html** - 完整HTML页面
   - 可在浏览器中预览效果
   - 用于本地测试

## 🚀 快速使用步骤

### 步骤 1: 复制代码
打开 `alibaba-code.html` 文件，复制全部代码

### 步骤 2: 粘贴到阿里国际站
1. 登录阿里国际站后台
2. 进入产品详情编辑页面
3. 找到"源代码"或"HTML编辑"按钮（通常在编辑器工具栏）
4. 点击切换到源代码模式
5. 粘贴复制的代码
6. 点击保存

### 步骤 3: 预览和发布
- 点击预览按钮查看效果
- 确认无误后发布

## ✏️ 内容修改指南

### 1. 修改标题
找到以下代码并修改：
```html
<h1 style="...">COMPANY PROFILE</h1>
<p style="...">Your Trusted Enterprise Storage Partner</p>
```

### 2. 修改公司介绍文字
找到 About Us 部分的 `<p>` 标签，直接修改文字内容：
```html
<p style="...">
    这里是公司介绍文字，可以直接修改
</p>
```

### 3. 修改统计数据
找到统计数据部分，修改数字和描述：
```html
<div style="color: #2874c9; font-size: 56px; ...">13+</div>
<div style="color: #555555; font-size: 18px; ...">Years Experience</div>
```

可以修改为：
- 数字：13+ → 20+
- 描述：Years Experience → 年经验

### 4. 修改优势项目
找到 Our Advantages 部分，修改每个优势的标题：
```html
<h3 style="...">High Quality<br>Products</h3>
```

### 5. 替换产品图片

**方法一：使用阿里云图片（推荐）**
1. 在阿里国际站后台上传图片
2. 复制图片链接
3. 替换代码中的图片地址：
```html
<img src="替换为您的图片链接" alt="Product Image 1" style="...">
```

**方法二：使用外部图片链接**
```html
<img src="https://您的图片链接.jpg" alt="Product Image 1" style="...">
```

## 🎨 样式修改指南

### 1. 修改主题色（蓝色）
当前主题色：#2874c9（商务蓝）

替换所有的 `#2874c9` 为您想要的颜色：
- 红色：#e74c3c
- 绿色：#27ae60
- 橙色：#e67e22
- 紫色：#9b59b6

找到标题部分的渐变背景：
```html
background: linear-gradient(135deg, #1e5ba8 0%, #2874c9 50%, #1a4d8f 100%);
```

可以使用在线渐变色生成器生成新的渐变色代码

- 主标题：`font-size: 48px;`
- 副标题：`font-size: 24px;`
- 正文：`font-size: 18px;`

直接修改数字即可

### 4. 修改间距
- 内边距：`padding: 40px;`
- 外边距：`margin-bottom: 40px;`
- 左右间距：`gap: 20px;`

## 💡 常见问题

### Q1: 代码粘贴后显示乱码？
**A:** 确保在"源代码"模式下粘贴，不要在可视化编辑模式粘贴

### Q2: 图片不显示？
**A:** 检查图片链接是否正确，建议使用阿里云OSS上传图片后获取链接

### Q3: 手机端显示不正常？
**A:** 阿里国际站会自动适配手机端，如果有问题，可以调整：
```html
max-width: 1200px; → max-width: 100%;
```

### Q4: 想要添加更多图片？
**A:** 复制图片展示的代码块，修改图片链接即可：
```html
<div style="flex: 1; text-align: center;">
    <img src="新图片链接" alt="Product Image 3" style="...">
    <p style="...">Product Display 3</p>
</div>
```

### Q5: 如何调整图片大小？
**A:** 修改图片链接中的尺寸参数，或在 img 标签中添加：
```html
<img src="..." style="width: 100%; max-width: 500px; height: auto; ...">
```
