## 1. 产品型号识别指南

### 1.1 服务器内存 (Server Memory)

#### 型号特征
服务器内存型号通常包含以下关键字：

| 品牌 | 型号前缀示例 | 识别特征 |
|------|-------------|----------|
| **Samsung三星** | M393A, M386A, M471A | 以 "M" 开头 |
| **SK Hynix海力士** | HMA, HMAA, HMT | 以 "HM" 开头 |
| **Micron美光** | MTA36ASF, MTA18ASF | 以 "MTA" 开头 |
| **Kingston金士顿** | KSM, KTH, KTD, KTL | 以 "K" 开头 |

#### DDR代数识别

```
DDR3: PC3-xxxxx (如 PC3-12800)
DDR4: PC4-xxxxx (如 PC4-21300, PC4-25600)
DDR5: PC5-xxxxx (如 PC5-38400)
```

#### 完整型号解析示例

**Samsung型号: M393A4K40CB2-CTD**
```
M393  - 产品系列（服务器RDIMM）
A     - DDR4
4K40  - 容量32GB (4Gbit x 4rank x 40chips)
CB    - 频率代码
2     - 版本
CTD   - 速度等级 2666MHz
```

**常见容量对照:**
- 8GB, 16GB, 32GB, 64GB, 128GB, 256GB

#### 内存类型
| 类型 | 全称 | 用途 |
|------|------|------|
| RDIMM | Registered DIMM | 服务器标准配置 |
| LRDIMM | Load-Reduced DIMM | 大容量服务器 |
| UDIMM | Unbuffered DIMM | 工作站/入门服务器 |
| NVDIMM | Non-Volatile DIMM | 持久性内存 |

## 3. 国际贸易条款详解 (Incoterms 2020)

### 3.1 常用贸易条款对比

| 条款 | 全称 | 风险转移点 | 卖方责任 |
|------|------|-----------|----------|
| **EXW** | Ex Works | 卖方工厂 | 最小 |
| **FOB** | Free On Board | 装上运输工具 | 出口清关+装货 |
| **CIF** | Cost, Insurance, Freight | 装上运输工具 | +运费+保险 |
| **DDP** | Delivered Duty Paid | 买方指定地点 | 最大(含进口关税) |
| **DAP** | Delivered at Place | 买方指定地点 | 不含进口关税 |

### 3.2 服务器配件贸易常用条款

#### EXW (工厂交货)
```
适用场景：买方有自己的货代
卖方责任：准备好货物
买方责任：所有运输、清关、保险
报价示例：USD 100.00/pc EXW Shenzhen
```

#### FOB (离岸价)
```
适用场景：最常用的出口条款
卖方责任：出口清关、送到机场/港口
买方责任：运费、进口清关
报价示例：USD 105.00/pc FOB Hong Kong
```

#### CIF (到岸价)
```
适用场景：买方希望卖方安排运输
卖方责任：运费+保险+出口清关
买方责任：进口清关、关税
报价示例：USD 120.00/pc CIF New York
```

#### DDP (完税交货)
```
适用场景：买方要求"门到门"
卖方责任：所有费用包含到买方仓库(含关税)
买方责任：无
报价示例：USD 135.00/pc DDP Los Angeles
风险提示：需了解目的国关税税率
```

### 3.3 选择贸易条款的考量

```
┌────────────────────────────────────────────┐
│         选择贸易条款决策树                   │
├────────────────────────────────────────────┤
│                                            │
│  Q1: 客户有指定货代吗？                     │
│      │                                     │
│      ├── 是 → EXW 或 FOB                   │
│      │                                     │
│      └── 否 → Q2: 客户要求含运费吗？        │
│                  │                         │
│                  ├── 是 → CIF 或 DAP/DDP   │
│                  │                         │
│                  └── 否 → FOB              │
│                                            │
│  Q3: 客户要求含关税吗？                     │
│      │                                     │
│      ├── 是 → DDP (需计算关税)             │
│      │                                     │
│      └── 否 → DAP 或 CIF                   │
│                                            │
└────────────────────────────────────────────┘
```

## 4. 空运快递公司及区别

### 4.1 国际快递公司对比

| 快递公司 | 优势 | 劣势 | 适用场景 |
|----------|------|------|----------|
| **DHL** | 速度快、网络广、服务好 | 价格最贵 | 紧急件、欧洲地区 |
| **FedEx** | 美洲线路强、速度快 | 价格较贵 | 美国/美洲、时效要求高 |
| **UPS** | 美国本土强、包裹追踪好 | 偏远地区弱 | 美国、大件货物 |
| **TNT** | 欧洲线路强 | 已被FedEx收购 | 欧洲地区 |

### 4.2 价格与时效对比

#### 时效参考（以中国发美国为例）

| 服务类型 | DHL | FedEx | UPS | 时效 |
|----------|-----|-------|-----|------|
| 特快 | Express | Priority | Express | 1-2天 |
| 标准 | Express | Economy | Expedited | 3-5天 |
| 经济 | - | Economy | Standard | 5-7天 |

#### 价格区间参考

```
以1KG货物，中国→美国为例（2024年参考价）：

DHL Express:     约 ¥180-250/kg
FedEx Priority:  约 ¥160-220/kg
FedEx Economy:   约 ¥80-120/kg
UPS Express:     约 ¥150-200/kg
UPS Expedited:   约 ¥90-130/kg
```

### 4.3 货代/专线选择

#### 香港线路优势
- 无需出口报关（深圳→香港→国外）
- 时效快
- 适合敏感货物
- 通常比直发便宜

#### 常见货代类型

| 类型 | 说明 | 价格 | 时效 |
|------|------|------|------|
| **国际快递账号** | DHL/FedEx/UPS官方账号 | 贵 | 快 |
| **快递代理** | 代理价格，走快递渠道 | 中等 | 快 |
| **空运专线** | 货代包机/拼柜 | 较便宜 | 中等 |
| **邮政EMS** | 国际邮政系统 | 便宜 | 慢 |

### 4.4 选择快递的考量因素

```
┌─────────────────────────────────────────────────────┐
│              快递选择决策矩阵                         │
├─────────────────────────────────────────────────────┤
│                                                     │
│  1. 目的地国家/地区                                  │
│     • 美国 → FedEx/UPS 优先                         │
│     • 欧洲 → DHL/TNT 优先                           │
│     • 东南亚 → DHL/顺丰国际                          │
│     • 其他 → 根据货代建议                            │
│                                                     │
│  2. 货物重量/体积                                    │
│     • <0.5kg → 小包专线最划算                        │
│     • 0.5-21kg → 快递                               │
│     • >21kg → 考虑空运专线                          │
│                                                     │
│  3. 时效要求                                        │
│     • 1-3天 → 快递特快                              │
│     • 3-7天 → 快递标准/经济                          │
│     • 7天以上 → 空运专线                            │
│                                                     │
│  4. 预算考量                                        │
│     • 预算充足 → DHL/FedEx优先                      │
│     • 预算有限 → 货代专线                           │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 4.5 运费计算方式

#### 实重 vs 体积重
```
实际重量 (Actual Weight)：货物实际称重

体积重量 (Volumetric Weight)：
  • 快递公司: 长×宽×高(cm) ÷ 5000
  • 空运货代: 长×宽×高(cm) ÷ 6000

计费重量 = MAX(实重, 体积重)
```

#### 计算示例
```
货物：2个服务器内存
实重：0.3kg
包装尺寸：25cm × 15cm × 10cm

体积重 = 25 × 15 × 10 ÷ 5000 = 0.75kg

计费重量 = 0.75kg（取较大值）
```

## 附录：常见问题FAQ

### Q1: 客户只给了P/N但我不确定是什么产品？
**A:** 可以通过以下方式确认：
1. 直接问客户 "Could you confirm what type of product this is?"
2. Google搜索型号
3. 查询供应商系统
4. 根据本指南的型号规则判断

### Q2: 客户要求的型号查不到？
**A:** 可能原因：
1. 型号输入有误，请客户再次确认
2. OEM定制型号，问客户是什么品牌服务器用的
3. 过于老旧的产品，已停产

### Q3: 如何判断报价是否合理？
**A:**
1. 对比多家供应商报价
2. 参考eBay、Amazon价格
3. 了解产品新旧程度
4. 考虑市场供需情况

### Q4: 客户要求DDP报价，如何计算？
**A:** DDP价格 = 产品成本 + 利润 + 运费 + 目的国关税 + 目的国增值税
- 需要查询HS Code和税率
- 建议找货代协助计算

### Q5: 发货后客户说没收到怎么办？
**A:**
1. 查询快递官网追踪状态
2. 联系快递公司客服查询
3. 如丢失，启动索赔流程
4. 与客户协商解决方案
