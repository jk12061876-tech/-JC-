815101-B21 64GB DDR4-2666MT/s LRDIMM ECC Server Memory

Product Overview

The 815101-B21 is a 64GB DDR4-2666MT/s (PC4-21328) LRDIMM ECC server memory module designed for enterprise data center servers. This 64GB Quad Rank x4 (4Rx4) DDR4-2666 ECC LRDIMM Server RAM features 288-Pin DDR4 configuration with PC4-21300 speed rating and 1.2V low power consumption. Operating at CL19 latency (CAS-19-19-19), this 64GB DDR4 server memory delivers reliable performance for demanding server workloads and server RAM upgrades.
